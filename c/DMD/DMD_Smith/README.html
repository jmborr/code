Discontinuous molecular dynamics code

The following FORTRAN codes are example programs for efficiency techniques in the simulation of hard chain molecules. See the reference "Discontinuous molecular dynamics for polymeric fluids.", J. Comput. Phys.", 1995.

DMD code using single-event scheduling, neighbor lists, link lists, and a simple binary tree for scheduling.

DMD code using multi-event scheduling, subcell method, and a doubly-linked binary tree for scheduling.

Use your Web browser to save the files as text.

Both of these codes need the header file which will be inserted when preprocessed by CPP. This will usually happen automatically for most FORTRAN compilers if the program file is given a .F extension. A sample configuration file for 32 chains of 16-mers is also provided with its input parameter file. A secondary file input file for square well simulations is also provided.
